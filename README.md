# 🌸 翻譯小助手 Translator App

這是一個使用 Python + Streamlit 製作的可愛風多語言翻譯工具，支援中英日韓法德互譯、翻譯複製、即時反應，還會慢慢變得更可愛 💖

## ✨ 功能特色

- 中文 ↔ 英文、日文、韓文、法文、德文
- 自動偵測語言
- 一鍵翻譯＋複製
- 介面簡單清新、適合自己使用

## 🚀 如何執行

```bash
pip install -r requirements.txt
streamlit run translator_app.py
```
